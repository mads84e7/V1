Hvad indeholder body?

Brødtekstelementet definerer dokumentets brødtekst og er en beholder til alt det synlige indhold, f.eks. Overskrifter, afsnit, billeder, hyperlinks, tabeller, lister osv.

Hvad kan head blandt andet indeholde og hvad SKAL det indeholde?

The head element contains meta information about the HTML page
